# 剪刀石頭布

###### tags: `RockPaperScissors`

### 初始化

`python >= 3.8`

```cmd=
cd ./python/RockPaperScissors
python -m venv env
env\Scripts\activate # or `source env/bin/activate`
pip install -r requirements.txt
python index.py
```
